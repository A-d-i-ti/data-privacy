## **ANONYMITY :**

| Technique           | Description                                                                          |
| ------------------- | ------------------------------------------------------------------------------------ |
| **Interactive**     | Requires real-time communication (e.g., mixers, MPC, CoinJoin).                      |
| **Non-Interactive** | One-shot or stand-alone cryptographic techniques (e.g., zk-SNARKs, Ring Signatures). |
